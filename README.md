# Jornadas de puertas abiertas 2025 - Adaptación al Ambiente de Trabajo 7mo I

Sitio web de conversión de unidades físicas en javascript para la materia Adaptación al Ambiente de trabajo y Física de 7 I.

"El objetivo de la materia es solucionar problemas de software en el ámbito laboral, es por ellos que a veces se debe elegir empezar de nuevo un proyecto o modificar lo existente."

"El proyecto consistió en generar un sitio web que permita la conversión de unidades físicas para que los estudiantes puedan tener una validación de sus cálculos dentro o fuera de clases".
